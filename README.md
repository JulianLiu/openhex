# OpenSlay

HTML5 game inspired by Slay.

Demo here: https://alcalyn.github.io/openslay/


## Install

``` bash
git clone git@github.com:alcalyn/openslay.git
cd openslay/

npm install
npm start
```


## Deploy

Deploy to `gh-pages`:

``` bash
npm run deploy
```
