# OpenHex

HTML5 game inspired by Slay.

Demo here: https://alcalyn.github.io/openhex/


## Install

``` bash
git clone git@github.com:alcalyn/openhex.git
cd openhex/

npm install
npm start
```


## Deploy

Deploy to `gh-pages`:

``` bash
npm run deploy
```
